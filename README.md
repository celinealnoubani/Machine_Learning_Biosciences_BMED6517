# BMED 6517 - Machine Learning for Biosciences

**Georgia Institute of Technology**

## Overview

This repository contains coursework for Machine Learning for Biosciences, covering machine learning techniques applied to biological and biomedical data analysis.

## Course Topics

- Supervised Learning
- Unsupervised Learning
- Deep Learning for Biomedical Applications
- Feature Engineering for Biological Data
- Model Evaluation and Validation

## Technologies

- Python
- Scikit-learn
- TensorFlow/PyTorch
- Pandas/NumPy
